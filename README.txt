Positive Pflege Pro – Hinweise

Installation
1. ZIP-Datei in WordPress unter Design > Themes > Theme hinzufügen > Theme hochladen hochladen.
2. Theme aktivieren.
3. Optional unter Einstellungen > Lesen die Startseite auf eine statische Seite setzen und die Startseite auswählen.
4. Unter Design > Customizer die Inhalte pflegen.
5. Menüs unter Design > Menüs anlegen und den Positionen "Hauptmenü" und "Footer-Menü" zuweisen.
